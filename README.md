## 🔄 Workflow collaboratif

Ce repo **privé** gère la gouvernance et la traçabilité. Les notes elles-mêmes vivent dans **Obsidian** (local + Publish public).

### Pour les collaborateurs

**Avant d'éditer une note importante dans Obsidian :**

1. Ouvrir une **Issue** (template : `modification.md`)
2. Décrire l'intention : quelle note, pourquoi, quoi
3. Attendre validation de Fred
4. Éditer dans Obsidian
5. Fred relit, valide et publie
6. Issue fermée

**Pour les petites corrections :** typos, formattage → pas d'Issue requise.

👉 **Lire en détail :** [`POLITIQUE_TRACABILITE.md`](./POLITIQUE_TRACABILITE.md)
